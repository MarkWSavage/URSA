# Changelog

## [Unreleased]

### Added

- Chord length distribution estimator: interactive ImGui/ImPlot GUI
  implementing Messenger & Ash Eq. 1.38 (Approximation No. 2) for a
  rectangular parallelepiped.
- CMake build with `FetchContent` for GLFW, Dear ImGui, and ImPlot.
- README with formula background, build/run instructions, screenshot,
  description, and topics.
- MIT license, with a README badge.
- `.gitattributes` marking PNGs as binary.
- GitHub Actions CI workflow building the project on push and pull
  request to `master`, with separate README status badges for each.
- `CONTRIBUTING.md` with setup and PR guidelines.
- `CODEOWNERS` assigning repo ownership.
